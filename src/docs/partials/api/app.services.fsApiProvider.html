fsApiProvider
service in module app

Description

This provider is a wrapper around built in $http that provides service that allows to interact with REST API servers in a simple and convenient way.

Methods

  • option(name, value)

    Getter/Setter. Provides ability either to obtain value of specified config property or set it (if second arg is provided)

    Parameters
    ParamTypeDetails
    namestring

    The name of config property. See options for avail properties.

    value*

    The value to assign to config property

    Example

    fsApiProvider.option('url', 'http://some.webserver.com/api/');

  • options(options)

    Allows you to preliminary configure all instances returned by provider.

    Parameters
    ParamTypeDetails
    optionsobject

    The object every property of which represents some aspect of configuration

    PropertyTypeDetails
    urlstring

    The base URL of the API

    timeoutinteger

    The number of milliseconds until the request timesout

    encodingstring

    The encoding of the request. Avail options:

    • url
    • json
    • formdata
    dataKeystring

    The key that represents the the data object in the response

    eventsobject

    Provides ability to assign custom callback that will be executed on specific stages\phases of ajax request

    • The start of ajax call
    • Ajax returns code in the 200 range
    • Upon the ajax completion
    • Ajax returns and codes other then the 200 range

    Example

    fsApiProvider.options({url: 'http://some.webserver.com/api/'});
